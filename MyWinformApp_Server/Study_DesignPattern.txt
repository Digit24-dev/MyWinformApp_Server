
	

	 목적
	   1. 생성 : 객체 인스턴스 생성에 관여, 클래스 정의와 객체 생성 방식을 구조화, 캡슐화를 수행하는 패턴
		[ Singleton Pattern ]
		
		: 유일한 하나의 인스턴스를 보장하도록 하는 패턴

		클래스를 정적으로 선언하여 메모리에 상주할 수 있게 생성한다.
		데이터만 사용하는 클래스를 싱글톤으로 선언할 수 있다.
		메모리를 적게 사용하는 클래스로 사용한다
		한 클래스에 한 객체만 존재하도록 제한.

		[ Builder ]
		: 복잡한 인스턴스를 조립하여 만드는 구조, 복합 객체를 생성할 때 객체를 생성하는 방법과 객체를 구현하는 방법을 분리함으로써 동일한 생성 절차에서 서로 다른 표현 결과를 만들 수 있는 디자인 패턴
		생성과 표기를 분리해서 복잡한 객체를 생성
